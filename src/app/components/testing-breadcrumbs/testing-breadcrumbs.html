Testing Breadcrumbs

The following demos showcase the app-qld-breadcrumb component with various configurations following the Queensland Government Design System.

1. Simple Breadcrumb (Two Items)

A basic two-item breadcrumb trail showing Home and the current page. The simplest form of navigation context.

<app-qld-breadcrumb [items]="simpleBreadcrumbs"></app-qld-breadcrumb>

2. Breadcrumb with Home Icon

Breadcrumbs prefixed with a home icon link for quick return to the homepage. Useful when space allows and visual cues are preferred.

<app-qld-breadcrumb [items]="withHomeIconBreadcrumbs"
  [showHomeIcon]="true"></app-qld-breadcrumb>

3. Three-Level Deep Breadcrumb

A three-level breadcrumb trail showing a typical content hierarchy: Home > Section > Subsection.

<app-qld-breadcrumb [items]="threeLevelBreadcrumbs"></app-qld-breadcrumb>

4. Four-Level Deep Breadcrumb

A four-level breadcrumb trail for deeper content hierarchies, common in documentation or complex navigation structures.

<app-qld-breadcrumb [items]="fourLevelBreadcrumbs"></app-qld-breadcrumb>

5. Five-Level Deep Breadcrumb

A maximum depth breadcrumb trail demonstrating how the component handles deeply nested pages. Items wrap naturally on smaller screens.

<app-qld-breadcrumb [items]="fiveLevelBreadcrumbs"></app-qld-breadcrumb>

6. Alt Variant (Dark Background)

Breadcrumbs styled for dark backgrounds using the alt variant. White text with lighter separators, designed for use on dark banner sections.

<app-qld-breadcrumb [items]="altBreadcrumbs"
  variant="alt"></app-qld-breadcrumb>

7. Breadcrumb Without Home Link

Breadcrumbs that start from a non-Home section. All items are linked except the current page, which is marked with aria-current.

<app-qld-breadcrumb [items]="noHomeBreadcrumbs"></app-qld-breadcrumb>

8. Breadcrumb with FontAwesome Icons

Breadcrumb items can include FontAwesome icons alongside text for visual distinction between navigation levels.

<app-qld-breadcrumb [items]="iconBreadcrumbs"></app-qld-breadcrumb>

9. Single Item (Current Page Only)

A breadcrumb with only the current page item. Useful for top-level pages where no navigation trail is needed beyond identifying the current location.

<app-qld-breadcrumb [items]="singleItemBreadcrumbs"></app-qld-breadcrumb>

10. Partial Navigation Trail

A breadcrumb where some items have hrefs and others don't. Items without hrefs render as plain text, useful for intermediate sections that aren't directly linkable.

<app-qld-breadcrumb [items]="partialBreadcrumbs"></app-qld-breadcrumb>
