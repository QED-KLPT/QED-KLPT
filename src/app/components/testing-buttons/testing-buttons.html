Testing CTA Links

The following demos showcase the app-cta-link component with various variants, states, and configurations following the Queensland Government Design System.

1. Default CTA Link

A simple and clear link to direct users to a preferred action. The default CTA uses bold text with an arrow icon that animates on hover.

Sign up 
<app-cta-link href="#">Sign up</app-cta-link>

2. View All CTA Link

When a CTA link appears at the end of link columns or dropdown menus, it uses a modified icon style with an arrow encircled in a circle to be visually distinct from other links.

View all 
<app-cta-link href="#" variant="view-all">
  <span class="qld__cta-link--view-all-icon--wrapper">View all</span>
</app-cta-link>

3. CTA in a List Context

CTA links are commonly used within link lists to guide users toward a preferred action. The arrow icon provides visual direction.

  • Find out more
  • Apply now
  • Contact us
<ul style="list-style: none; padding: 0;">
  <li><app-cta-link href="#">Find out more</app-cta-link></li>
  <li><app-cta-link href="#">Apply now</app-cta-link></li>
  <li><app-cta-link href="#">Contact us</app-cta-link></li>
</ul>

4. CTA with Leading Icon

A CTA link can include an icon before the text for additional visual context. This is useful when the action needs a clear visual cue alongside the text.

Subscribe to newsletter 
<app-cta-link href="#">
  <i class="fa-light fa-envelope" aria-hidden="true"></i>
  Subscribe to newsletter
</app-cta-link>

5. Disabled CTA Link

A disabled CTA link uses reduced opacity and a not-allowed cursor. Used when an action is temporarily unavailable, such as before form validation passes.

Unavailable action 
<app-cta-link href="#" [disabled]="true">Unavailable action</app-cta-link>

6. CTA on Dark Background

CTA links automatically adapt their colors when placed on a dark background. The link text and arrow icon switch to light-colored tokens for proper contrast.

Learn more 
<div class="qld__body--dark" style="padding: 1.5rem;">
  <app-cta-link href="#">Learn more</app-cta-link>
</div>

7. CTA with aria-label (Accessibility)

When the link text alone doesn't fully communicate the destination, an aria-label provides additional context for screen reader users.

Sign up 
<app-cta-link href="/signup"
  aria-label="Sign up for our monthly newsletter">
  Sign up
</app-cta-link>

8. CTA Opening in New Tab

CTA links can open in a new browser tab using the target attribute. This is appropriate for external resources or when you want to keep the current page open.

Visit qld.gov.au (opens in new tab) 
<app-cta-link href="https://www.qld.gov.au"
  target="_blank">Visit qld.gov.au</app-cta-link>

9. View All in a Navigation List

The View All variant is designed specifically for use at the end of navigation lists or link columns. The encircled arrow makes it visually distinct from regular list items.

  • Programs
  • Services
  • Resources
  • View all
<ul style="list-style: none; padding: 0;">
  <li><app-cta-link href="#">Programs</app-cta-link></li>
  <li><app-cta-link href="#">Services</app-cta-link></li>
  <li><app-cta-link href="#">Resources</app-cta-link></li>
  <li>
    <app-cta-link href="#" variant="view-all">
      <span class="qld__cta-link--view-all-icon--wrapper">View all</span>
    </app-cta-link>
  </li>
</ul>

10. Multiple CTA Variants Side by Side

Default and View All variants can be used together to show the visual difference between standard CTA links and navigation-focused "View all" links.

Default:

Sign up

View All:

View all 
<!-- Default -->
<app-cta-link href="#">Sign up</app-cta-link>

<!-- View All -->
<app-cta-link href="#" variant="view-all">
  <span class="qld__cta-link--view-all-icon--wrapper">View all</span>
