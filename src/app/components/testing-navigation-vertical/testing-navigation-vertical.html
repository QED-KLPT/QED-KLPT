Testing Vertical Navigation

The following demos showcase the app-vertical-nav component with various configurations following the Queensland Government Design System vertical navigation patterns.

1. Basic Links Only

A simple flat list of navigation links without icons, submenus, or headers. Used for straightforward single-level navigation.

<app-vertical-nav [items]="basicLinks"></app-vertical-nav>

2. With Icons

Navigation links with Font Awesome icons alongside labels for visual identification. Icons use the qld__left-nav__item-icon class styling.

<app-vertical-nav [items]="withIcons"></app-vertical-nav>

3. Accordion Submenus

Expandable and collapsible nested navigation sections. Click the chevron button to toggle submenu visibility. Uses aria-expanded for accessibility.

<app-vertical-nav [items]="accordionSubmenus"></app-vertical-nav>

4. Active Page State

Navigation with the current page highlighted using a left border accent and purple background. The active item receives aria-current="page" styling.

<app-vertical-nav [items]="activePageState"></app-vertical-nav>

5. Multi-Level Nesting

Three-level deep submenu hierarchy demonstrating nested expandable sections. Each level has appropriate indentation and chevron controls.

<app-vertical-nav [items]="multiLevelNesting"></app-vertical-nav>

6. Section Headers

Non-clickable section dividers that group related navigation items. Headers use uppercase styling with letter-spacing for visual separation.

<app-vertical-nav [items]="sectionHeaders"></app-vertical-nav>

7. Mixed Active + Submenu

Combination of active state and expandable sections. The Dashboard section is both the current page and expanded to show its children.

<app-vertical-nav [items]="mixedActiveSubmenu"></app-vertical-nav>

8. With Title/Header

Navigation panel with a heading label at the top. Useful for contextual navigation within a specific section of the application.

<app-vertical-nav title="Design System"
  [items]="withTitle"></app-vertical-nav>

9. Compact/Slim Variant

Reduced spacing for dense navigation with fewer items. Ideal for sidebars where space is limited and quick access to key pages is needed.

<app-vertical-nav [items]="compactNav"></app-vertical-nav>

10. Dark Theme Variant

Navigation with dark background styling using the KLPT purple theme. Demonstrates how the component adapts to different color schemes via CSS variables.

<div style="background: var(--doe-color-heading);">
  <app-vertical-nav [items]="darkThemeNav"></app-vertical-nav>
</div>
