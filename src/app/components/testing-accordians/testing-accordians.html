Testing Accordians

The following demos showcase the app-qld-accordion-group and app-qld-accordion-item components with various attributes and configurations.

1. Basic Single Accordion

A single accordion item in a group, default closed state.

Here is some accordion content. This demonstrates a basic single accordion item.

 
<app-qld-accordion-group>
  <app-qld-accordion-item title="Accordion heading 1">
    <p>Here is some accordion content...</p>
  </app-qld-accordion-item>
</app-qld-accordion-group>

2. Pre-expanded Accordion

An accordion item that starts in the expanded state on page load.

This accordion is already open when the page loads. Useful for highlighting important content.

 
<app-qld-accordion-item title="Accordion heading 1" [expanded]="true">
  <p>This accordion is already open...</p>
</app-qld-accordion-item>

3. Dark Variant Accordion

An accordion item with the dark background variant applied.

This accordion uses the dark variant styling. The background and text colors are adjusted for dark themes.

 
<app-qld-accordion-item title="Dark accordion heading" [dark]="true">
  <p>This accordion uses the dark variant...</p>
</app-qld-accordion-item>

4. Alt Background Variant

An accordion item with the alt background variant for different visual contexts.

This accordion uses the alt variant styling. It provides a different visual treatment suitable for alternate content sections.

 
<app-qld-accordion-item title="Alt background accordion" [alt]="true">
  <p>This accordion uses the alt variant...</p>
</app-qld-accordion-item>

5. Accordion Group with Multiple Items

A group of three accordions where only one panel can be open at a time.

Content for accordion item one. Clicking this will close any other open panel.

Content for accordion item two. Only one panel remains open at a time in a group.

Content for accordion item three. The group manages the open/close state coordination.

 
<app-qld-accordion-group>
  <app-qld-accordion-item title="Accordion heading 1">...</app-qld-accordion-item>
  <app-qld-accordion-item title="Accordion heading 2">...</app-qld-accordion-item>
  <app-qld-accordion-item title="Accordion heading 3">...</app-qld-accordion-item>
</app-qld-accordion-group>

6. Group with "Open All" Toggle

An accordion group with the toggle all button enabled. Shows when 3 or more items are present.

Click the "Open all" button above to expand every panel at once.

Click "Close all" to collapse every panel. The button label toggles automatically.

You can also click individual headers to toggle specific panels while others remain open.

 
<app-qld-accordion-group [toggleAll]="true">
  <app-qld-accordion-item title="Accordion heading 1">...</app-qld-accordion-item>
  <app-qld-accordion-item title="Accordion heading 2">...</app-qld-accordion-item>
  <app-qld-accordion-item title="Accordion heading 3">...</app-qld-accordion-item>
</app-qld-accordion-group>

7. Dark Group

An entire accordion group with the dark background variant applied to all items.

This entire group uses dark styling. All items inherit the dark theme.

The dark variant provides higher contrast for use on light page backgrounds.

Each item in the dark group maintains proper accessibility with ARIA attributes.

 
<app-qld-accordion-group [dark]="true">
  <app-qld-accordion-item title="Dark group heading 1">...</app-qld-accordion-item>
  <app-qld-accordion-item title="Dark group heading 2">...</app-qld-accordion-item>
  <app-qld-accordion-item title="Dark group heading 3">...</app-qld-accordion-item>
</app-qld-accordion-group>

8. Pre-expanded First Item

A group where the first item starts expanded while others remain closed.

This first item is pre-expanded. Clicking other items will close this one.

This item starts closed. Clicking it will expand this panel and close the first.

This item also starts closed. Only one panel can be open at a time.

 
<app-qld-accordion-item title="First item expanded" [expanded]="true">...</app-qld-accordion-item>

9. Mixed Dark and Alt Variants

A group with individual items using different variants (dark, alt, default).

This item uses the dark background variant.

This item uses the alt background variant.

This item uses the default styling.

 
<app-qld-accordion-item title="Dark variant item" [dark]="true">...</app-qld-accordion-item>
<app-qld-accordion-item title="Alt variant item" [alt]="true">...</app-qld-accordion-item>
<app-qld-accordion-item title="Default variant item">...</app-qld-accordion-item>

10. Accordion with Rich Content

An accordion containing links, lists, and formatted text to demonstrate content flexibility.

This accordion contains a hyperlink and formatted text.

  • First list item
  • Second list item with bold text
  • Third list item

You can include any HTML content inside an accordion panel.

Sub-heading in accordion

This demonstrates that headings and other block elements work inside accordions.
