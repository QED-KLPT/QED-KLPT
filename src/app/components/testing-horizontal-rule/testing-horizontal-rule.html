Testing Horizontal Rule

The following demos showcase the app-horizontal-rule component with various sizes and configurations following the Queensland Government Design System.

1. Small Horizontal Rule (Default)

The default small horizontal rule. Used to separate elements within a container or component. Based on the general spacing between components like buttons or cards (32px).

<app-horizontal-rule size="sm"></app-horizontal-rule>

2. Medium Horizontal Rule

The medium horizontal rule. Used to separate content within a section, matching the spacing above an H2 element (48px).

<app-horizontal-rule size="md"></app-horizontal-rule>

3. Large Horizontal Rule

The large horizontal rule. Used to separate sections of content, matching the default space between two sections of the same colour (64px).

<app-horizontal-rule size="lg"></app-horizontal-rule>

4. Small on Light Background

The small horizontal rule rendered on a light background. Horizontal rules work on any background colour in the QLD Design System.

<div style="background: var(--doe-color-surface); padding: 1.5rem;">
  <app-horizontal-rule size="sm"></app-horizontal-rule>
</div>

5. Medium on Alt (Grey) Background

The medium horizontal rule rendered on an alt/grey background. Demonstrates the rule's adaptability across different surface colours.

<div style="background: var(--klpt-cream); padding: 1.5rem;">
  <app-horizontal-rule size="md"></app-horizontal-rule>
</div>

6. Large on Dark Background

The large horizontal rule rendered on a dark background. The QLD Design System horizontal rules are designed to work with the border colour variables of each palette.

<div style="background: var(--klpt-ink); padding: 1.5rem;">
  <app-horizontal-rule size="lg"></app-horizontal-rule>
</div>

7. Decorative Horizontal Rule (aria-hidden)

A decorative horizontal rule with aria-hidden="true". Used when the rule is purely visual separation without conveying meaning. Screen readers will ignore this element.

<app-horizontal-rule size="sm" [decorative]="true"></app-horizontal-rule>

8. Thematic Break (Screen Reader Announced)

A thematic break horizontal rule without aria-hidden. Used when the rule indicates a shift in content or topic. Screen readers will announce this as a thematic break.

<app-horizontal-rule size="md" [decorative]="false"></app-horizontal-rule>

9. All Three Sizes Stacked

All three horizontal rule sizes displayed together for comparison. Small (default), Medium, and Large — each with increasing line weight and vertical spacing.

Small (default) Medium Large 
<app-horizontal-rule size="sm"></app-horizontal-rule>
<app-horizontal-rule size="md"></app-horizontal-rule>

10. Horizontal Rule Between Content Blocks

A practical example showing horizontal rules used to separate content blocks, demonstrating the recommended usage pattern for visual organisation.

First content block — this could be a paragraph, a list, or any other content.

Second content block — the horizontal rule provides clear visual separation between sections.

Third content block — consistent spacing improves readability and information hierarchy.
