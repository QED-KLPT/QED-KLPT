Testing Alerts

The following demos showcase the app-qld-alert component with the four alert types, five palette states, link support, expandable content, and dismissible mode.

1. Basic Info Alert

Default appearance for general information.

This is a basic informational alert with standard styling. 
<app-qld-alert type="info" title="Information title">
  This is a basic informational alert.
</app-qld-alert>

2. Basic Success Alert

Confirms a successful action.

Your form has been submitted successfully. 
<app-qld-alert type="success" title="Success title">
  Your form has been submitted successfully.
</app-qld-alert>

3. Basic Error Alert

Indicates a blocking error (uses h4 heading per QLD spec).

Something went wrong. Please try again later. 
<app-qld-alert type="error" title="Error Title">
  Something went wrong. Please try again later.
</app-qld-alert>

4. Basic Warning Alert

Draws attention to something urgent.

Your password will expire soon. Please update it. 
<app-qld-alert type="warning" title="Warning">
  Your password will expire soon. Please update it.
</app-qld-alert>

5. Info Alert with Link

Alert containing a hyperlink for further information.

This is info link 
<app-qld-alert type="info" title="Information title">
  This is info <a href="#">link</a>
</app-qld-alert>

6. Success Alert with Link & Expandable Content

Success alert with a link and a clickable "Show more" toggle.

Record saved view details Additional details about the saved record can be found here. All changes have been persisted to the database. 
<app-qld-alert type="success" title="Success title" showMore>
  Record saved <a href="#">view details</a>
  <span qld-more>Additional details here...</span>
</app-qld-alert>

7. Error Alert with Detailed Message

Form validation errors or blocking errors with recovery suggestions.

The form contains 3 errors that must be corrected before you can proceed. Please review the fields marked in red. 
<app-qld-alert type="error" title="Error">
  The form contains 3 errors that must be corrected...
</app-qld-alert>

8. Critical Warning

Urgent warning about potential consequences.

Your storage is nearly full. You have less than 10% remaining. Please free up space or upgrade your plan. 
<app-qld-alert type="warning" title="Warning">
  Your storage is nearly full. You have less than 10% remaining...
</app-qld-alert>

9. Light State Info Alert

Light palette variant with a softer background.

An updated policy has been published. Please review the changes at your earliest convenience. 
<app-qld-alert type="info" title="Information title" state="light">
  An updated policy has been published...
</app-qld-alert>

10. Dark State Warning

Dark palette variant for alerts on light backgrounds that need higher contrast.

Server maintenance scheduled for tonight from 2:00 AM to 4:00 AM AEST. Services may be temporarily unavailable. 
<app-qld-alert type="warning" title="Warning" state="dark">
  Server maintenance scheduled for tonight...
</app-qld-alert>
