Testing Navigational Horizontal

The following demos showcase the <app-qld-main-nav> component with various configurations following the Queensland Government Design System horizontal navigation patterns.

1. Simple Light Navigation

A basic horizontal navigation bar with a home icon and simple text links. The active item is highlighted with a darker color. This is the most common pattern for sites with straightforward information architecture.

{{ example1Code }}

2. Dark Navigation

A dark-themed horizontal navigation bar using the Queensland Government Design System dark variant. Suitable for sites that use a dark primary color scheme or want higher contrast navigation.

{{ example2Code }}

3. Navigation with Icons

Horizontal navigation with Font Awesome icons preceding each link label. Icons help users quickly identify menu items and reduce cognitive load. Best used sparingly for the most important navigation categories.

{{ example3Code }}

4. Simple Dropdown Menu

A navigation item with a simple dropdown submenu containing child links. The submenu appears on hover for desktop and on click for mobile. Ideal for sites with a two-level information hierarchy.

{{ example4Code }}

5. Mega Menu (3-Column)

A mega menu variant that displays child links in a three-column grid layout. This pattern is useful when a navigation category has many subpages that need to be organized into logical groups for easy scanning.

{{ example5Code }}

6. Mega Menu with Descriptions

A mega menu where each sub-link includes a short description paragraph beneath it. This provides additional context to help users understand what each link leads to before clicking, improving findability.

{{ example6Code }}

7. Dark Mega Menu

A dark-themed mega menu combining the dark navigation variant with a multi-column dropdown. Used when the entire site uses a dark color scheme and needs to display many subpages.

{{ example7Code }}

8. Navigation with Active States

Demonstrates how active items are styled differently from inactive ones. The active link uses a darker color to indicate the current page, helping users orient themselves within the site structure.

{{ example8Code }}

9. Extended Navigation with Multiple Dropdowns

A navigation bar with multiple items that have dropdown submenus. Each submenu operates independently, allowing users to navigate between different sections of the site without losing context.

{{ example9Code }}

10. Mixed Icons and Dropdowns

A combination navigation pattern that includes both icon-prefixed links and items with dropdown submenus. This flexible approach allows designers to use icons for visual emphasis while maintaining dropdown functionality for complex hierarchies.
