Testing Tables

The following demos showcase the Queensland Government Design System table component with various configurations and utility classes.

1. Basic Table

A simple table with the base qld__table class. The most fundamental table structure with caption, header, body, and footer.

@for (row of basicRows; track row.name) { }
Child Assessment OverviewTable is ordered by child name
Child Name Subject Area Progress Level
{{ row.name }} {{ row.subject }} {{ row.status }}
Total 5 children assessed — 
<table class="qld__table">
  <caption>Child Assessment Overview
    <span class="qld__caption">Table is ordered by child name</span>
  </caption>
  <thead>
    <tr>
      <th scope="col">Child Name</th>
      <th scope="col">Subject Area</th>
      <th scope="col">Progress Level</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>...</td><td>...</td><td>...</td></tr>
  </tbody>
  <tfoot><tr><td>...</td><td>...</td><td>...</td></tr></tfoot>
</table>

3. Striped/Banded Table

For tables with more rows, use the striped alternative. The qld__table--striped class applies alternating row colors to help the eye track across rows, especially helpful for readers with reading difficulties.

5. Custom Header Widths

Apply qld__table__header--width-N to individual <th> elements to set custom column widths. Available values: 10, 15, 20, 25, 33, 40, 50, 75 (percentages). Wrap in qld__table__wrapper with a grid class for horizontal scrolling on small screens.

@for (row of customWidthRows; track row.name) { }
Child Assessment OverviewTable is ordered by child name
Child Name Subject Area Progress Level
{{ row.name }} {{ row.subject }} {{ row.status }}
Total 5 children assessed — 
<div class="qld__table__wrapper col-xs-10">
  <table class="qld__table">
    <thead>
      <tr>
        <th scope="col" class="qld__table__header--width-25">Child Name</th>
        <th scope="col" class="qld__table__header--width-50">Subject Area</th>
        <th scope="col" class="qld__table__header--width-25">Progress Level</th>
      </tr>
    </thead>
    ...
  </table>
</div>

6. Multilevel Headings

Tables with multiple header rows use rowspan and colspan to create grouped column headers. Use qld__table__cell-left-border to add vertical dividers between grouped columns.

@for (row of multilevelRows; track row.childName) { }

Child Name Assessment Results
Domain Level
{{ row.category ? row.category : '' }} {{ row.childName }} {{ row.score }}
6 children assessed —

  <table class="qld__table qld__table--striped
    qld__table__col-2-left-border
    qld__table__col-3-left-border">
    <caption>Assessment Results by Domain</caption>
    <thead>
      <tr>
        <th rowspan="2" scope="col">Child Name</th>
        <th colspan="2" class="qld__table__cell-left-border qld__table__cell--middle">Assessment Results</th>
      </tr>
      <tr>
        <th scope="col" class="qld__table__cell-left-border">Domain</th>
        <th scope="col" class="qld__table__cell-left-border qld__table__cell--numeric">Level</th>
      </tr>
    </thead>
    ...
  </table>
</div>
