Testing Side Navigation

The following demos showcase the app-qld-side-nav component with various configurations following the Queensland Government Design System side navigation patterns.

1. Basic Flat List

A simple side navigation with a flat list of links and no nesting. Used for straightforward navigation structures.

<app-qld-side-nav
  [items]="flatListItems"
  title="In this section">
</app-qld-side-nav>

2. With Accordion Toggle

A side navigation with an accordion-style toggle button for mobile/collapsed state. The entire nav content can be collapsed on smaller screens.

<app-qld-side-nav
  [items]="withAccordionToggleItems"
  title="In this section">
</app-qld-side-nav>

3. Active State

A side navigation with one item marked as active using the active property. Active items are displayed as plain text (not links) with aria-current="page".

<app-qld-side-nav
  [items]="activeStateItems"
  title="In this section">
</app-qld-side-nav>

4. With Tick Icons

A side navigation with SVG tick icons on each link item, following the QLD Design System pattern for verified/complete items.

<app-qld-side-nav
  [items]="withTickIconsItems"
  title="In this section">
</app-qld-side-nav>

5. Two-Level Nesting

A side navigation with parent items that expand to show child links. Supports up to 3 levels of hierarchy as recommended by the QLD Design System.

<app-qld-side-nav
  [items]="twoLevelItems"
  title="In this section">
</app-qld-side-nav>

6. Three-Level Deep Nesting

A side navigation with three levels of nesting (grandparent → parent → child). This is the maximum recommended depth for side navigation.

<app-qld-side-nav
  [items]="threeLevelItems"
  title="In this section">
</app-qld-side-nav>

7. With Section Title

A side navigation with a section title rendered as an <h2> element with the qld__sidenav__title class, followed by navigation links.

<app-qld-side-nav
  [items]="withSectionTitleItems"
  title="In this section">
</app-qld-side-nav>

8. Mixed Links and Plain Items

A side navigation combining clickable links with plain text items (including an active state item). Plain items use <span> instead of <a>.

<app-qld-side-nav
  [items]="mixedLinksPlainItems"
  title="In this section">
</app-qld-side-nav>

9. Collapsed State

A side navigation that starts in a collapsed state with the [collapsed]="true" input. The accordion toggle button shows a closed chevron initially.

<app-qld-side-nav
  [items]="collapsedStateItems"
  title="In this section"
  [collapsed]="true">
</app-qld-side-nav>

10. Full Real-World Example

A complete side navigation matching the QLD Design System demo: section title, nested items with children, tick icons on leaf items, and an active state indicator.

<app-qld-side-nav
  [items]="fullRealWorldItems"
  title="In this section">
</app-qld-side-nav>
