Testing Tags

The following demos showcase the qld-tag component with various types, variants, and configurations following the Queensland Government Design System.

1. Default Tag

A basic non-interactive tag displayed as a pill shape. Used for simple categorization and labeling content.

Default tag 
<qld-tag>Default tag</qld-tag>

2. Large Default Tag

A larger default tag variant with increased font size and padding. Used when tags need more visual prominence.

Large default tag 
<qld-tag size="large">Large default tag</qld-tag>

3. Link Tag

An interactive tag styled as a link with transparent background and blue text. Clickable tags guide users to filtered content.

Link tag 
<qld-tag type="link" href="#">Link tag</qld-tag>

4. Large Link Tag

A large interactive link tag combining the link variant with increased size for emphasis in navigation contexts.

Large link tag 
<qld-tag type="link" size="large" href="#">Large link tag</qld-tag>

5. Info Tag (File Metadata)

An info tag with a light blue background and border. Commonly used for file metadata like format, size, or type indicators.

PDF, 2.4 MB 
<qld-tag type="info">PDF, 2.4 MB</qld-tag>

6. Large Info Tag (Reading Time)

A large info tag used for supplementary information like estimated reading time or content duration.

Approx. 5 min read 
<qld-tag type="info" size="large">Approx. 5 min read</qld-tag>

7. Filter Tag with Close Button

A filter tag that includes a close button for removing the applied filter. Clicking the X emits a close event.

Mathematics 
<qld-tag type="filter" closable (close)="handleClose()">Mathematics</qld-tag>

8. Mixed Tags in a List

A group of tags displayed within a tag list container. Useful for displaying multiple categories or labels together.

  • Language
  • Numeracy
  • Social
  • Physicality
<ul class="qld__tag-list">
  <li><qld-tag>Language</qld-tag></li>
  <li><qld-tag type="info">Numeracy</qld-tag></li>
  <li><qld-tag type="link" href="#">Social</qld-tag></li>
  <li><qld-tag size="large">Physicality</qld-tag></li>
</ul>

9. Filter Tag Wrapper with Title

A filter tag group wrapped under a "Filtered by:" heading. Common pattern for search and filter result pages.

Filtered by:

Reading Writing Speaking 
<h4>Filtered by:</h4>
<qld-tag type="filter" closable (close)="handleClose()">Reading</qld-tag>
<qld-tag type="filter" closable (close)="handleClose()">Writing</qld-tag>

10. Tags on Dark Background

Tags displayed within a dark background container. Demonstrates tag visibility and contrast on darker surfaces.

Link tag Info tag Large tag 
<div class="qld__body--dark" style="padding: 1.5rem;">
  <qld-tag type="link" href="#">Link tag</qld-tag>
  <qld-tag type="info">Info tag</qld-tag>
  <qld-tag size="large">Large tag</qld-tag>
</div>
