Testing Loading Spinner

The following demos showcase the app-loading-spinner component with various configurations following the Queensland Government Design System.

1. Default Vertical

The default loading spinner displays vertically with the spinner icon above the label text. This is the standard orientation for most use cases.

<app-loading-spinner label="Loading..."></app-loading-spinner>

2. Landscape (Horizontal)

The landscape variant displays the spinner icon inline with the label text, useful for compact layouts or inline loading states.

<app-loading-spinner label="Loading..." [landscape]="true"></app-loading-spinner>

3. Dark Theme

The dark theme variant is designed for use on dark backgrounds, with adjusted colors for the spinner wheel and label text.

<div style="background: #1b1b1b; padding: 2rem;">
  <app-loading-spinner label="Loading..." [dark]="true"></app-loading-spinner>
</div>

4. Icon Only

The icon-only variant hides the visible label text while maintaining accessibility through the role="status" attribute for screen readers.

<app-loading-spinner [iconOnly]="true"></app-loading-spinner>

5. Small Size

The small size variant reduces the spinner wheel to 1rem, suitable for use within buttons or compact UI elements.

<app-loading-spinner label="Loading..." [size]="'sm'"></app-loading-spinner>

6. Medium Size (Default)

The medium size is the default spinner wheel size at 2rem, providing a balanced appearance for most contexts.

<app-loading-spinner label="Loading..." [size]="'md'"></app-loading-spinner>

7. Large Size

The large size variant increases the spinner wheel to 3rem, ideal for prominent loading states or empty page placeholders.

<app-loading-spinner label="Loading..." [size]="'lg'"></app-loading-spinner>

8. Dark + Landscape

Combines the dark theme with landscape orientation for use on dark backgrounds where horizontal layout is preferred.

<div style="background: #1b1b1b; padding: 2rem;">
  <app-loading-spinner label="Processing..." [dark]="true" [landscape]="true"></app-loading-spinner>
</div>

9. Inside a Card

The loading spinner can be embedded within an app-card component to indicate loading states in card-based layouts.

<app-card title="Loading Data"
  description="Please wait..."
  type="no-action">
  <app-loading-spinner label="Fetching data..."></app-loading-spinner>
</app-card>

10. Centered in Container

The spinner can be centered within a container using flexbox utilities for full-page or section loading states.

<div style="display: flex;
  justify-content: center;
  align-items: center;
  min-height: 200px;">
  <app-loading-spinner label="Loading content..."></app-loading-spinner>
</div>
