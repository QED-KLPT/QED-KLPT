Executive function

Executive function is a set of skills that help children manage impulses, stay focused and work towards goals. These skills support learning and development across all areas and are shaped by each child’s cultural, social and emotional experiences.

Families are children’s first teachers, so executive function skills are influenced by family routines, relationships and community values. Recognising the whole child means seeing these relationships as central to their learning, wellbeing and identity. For Aboriginal and Torres Strait Islander children, awareness of their kinship networks and connections to Country are important to developing identity, which supports growth in executive function skills.

Overview

Children begin developing executive function in early childhood. It includes three key skills:

  • Working memory makes it possible to keep information in mind, such as remembering instructions long enough to carry them out.
  • Inhibitory control supports the ability to pause before acting and choose a suitable response.
  • Cognitive flexibility enables attention to shift and adapt from one task to another.

Executive function skills are difficult to observe because they are internal cognitive processes. Teachers and educators can see executive function in action through observable behaviours like persistence, adaptability and problem solving.

  • Persistence: staying on task, trying again after challenges and completing activities.
  • Adaptability: adjusting when routines change, responding to new information and staying calm under uncertainty.
  • Problem solving: exploring ideas, testing strategies and reflecting on what works.

These behaviours draw on different aspects of executive function, for example, persistence involves inhibitory control, adaptability reflects cognitive flexibility and problem-solving uses working memory.
